# 小红书内容风格

内容特点：
1. 标题要有真实痛点。
2. 开头要生活化，不要像广告。
3. 正文要分点清晰。
4. 多用“小白能看懂”的语言。
5. 内容适合收藏。
6. 结尾引导评论或私信，但不要强销售。

常用内容类型：
1. 小白科普型。
2. 避坑清单型。
3. 情绪共鸣型。
4. 家庭案例型。
5. 企业主场景型。
6. 品牌专业观点型。

标题风格：
- 第一次了解香港保险，先别急着问收益
- IUL 到底是什么？小白先看懂这 3 点
- 买保险前，真正要看懂的是这几个问题
- 成年人的安全感，很多时候来自提前规划
- 企业主家庭，为什么更需要提前做保障规划？

禁用标题：
- 买了就赚
- 稳赚不赔
- 闭眼买
- 高收益保险
- 普通人财富自由捷径
